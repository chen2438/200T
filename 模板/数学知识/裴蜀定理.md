## 裴蜀定理

对于整数`a,b`和正整数`x,y`，`ax+by=c`成立的**充要**条件是`gcd(a,b)%c=0`.

**推论**：`a,b`互质的**充要**条件是存在整数`x,y`，使`ax+by=1`.

**拓展**：对于`n`个整数`a1,a2,......,an`，`a1*x1+a2*x2+......+an*xn=S`成立的**充要**条件是`gcd(a1,a2,......an)%S=0`.

**`AC` 代码**

```cpp
#include<bits/stdc++.h>
#define FOR(i,a,b) for(int i=(a);i<=(b);++i)
using namespace std;

int main(){
    int n; cin>>n;
    int ans=0,t;
    FOR(i,1,n){
        scanf("%d",&t);
        t=abs(t);
        ans=__gcd(ans,t);
    }
    cout<<ans;
}
```
